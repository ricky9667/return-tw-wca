Return Taiwan WCA

Brings back Taiwan for WCA webpages.

Chrome Extension Firefox Add-on

Introduction

Due to this announcement from WCA, speedcubers from some specific regions are forced to be changed. Although it is difficult to revert this change, at least we can trick ourselves with this extension :P

Features

Return Taiwan WCA changes back the name for Taiwan, Hong Kong, and Macau, and changes the Chinese Taipei flag back the Taiwan flag. The changes will be applied to WCA and WCA Live.

Compatibility

The extension supports all Chromium browsers, Firefox add-on browsers, and Firefox Android.
