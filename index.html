Return Taiwan WCA

A web extension that changes Chinese Taipei back to Taiwan on all WCA results.

Chrome Extension Firefox Add-on

Introduction

Why Return Taiwan WCA?

The World Cube Association (WCA) lists Taiwan as Chinese Taipei in its results. This browser extension automatically changes Chinese Taipei back to Taiwan across all WCA pages, so Taiwanese competitors are properly represented.

How to Develop or Use Locally

Chrome

Chrome Extension
  1. Download or git clone this repository.
  2. Go to the Chrome extensions page: chrome://extensions/ (Remember to turn on developer mode, located on the top right of the screen)
  3. Click Load Unpacked and select the repository folder.

Firefox

Firefox Add-on
  1. Download or git clone this repository.
  2. Go to the Firefox debugging page: about:debugging#/runtime/this-firefox
  3. Click Load Temporary Add-on and select the manifest.json file from the repository folder.
